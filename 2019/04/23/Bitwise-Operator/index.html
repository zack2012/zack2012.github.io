位运算

发表于 2019-04-23 | 更新于 2019-06-25 | 分类于 编程语言 | 阅读次数：

在平时的工作中，位运算用的并不多，但在算法面试中，位运算是一种很重要的技巧。本文介绍一些常用的位运算技巧以及使用上需要注意的地方。因为c++的位运算来自c，所以下文中关于c++位运算的描述也同样使用于c和objective-c。

1. 位运算符

编程语言中一般都有以下6种位运算：

  • &(与运算)，真值表如下：
0 1
0 0 0
1 0 1
  • |(或运算)，真值表如下：
0 1
0 0 1
1 1 1
  • ^(异或运算)，相同为假，不同为真，真值表如下：
0 1
0 0 1
1 1 0

  • ~(取反)，按位取反，单目运算符。

  • <<(左移)，将一个整数向左移动n位。

  • >>(右移)，将一个整数向右移动n位。

  • 根据要求的位数左移或右移
  • 超出范围的位丢弃
  • 移动后空出的位置用0填充

这种位移也被称为逻辑位移.

  • 超出范围的位丢弃
  • 左移空出的位置用0填充，右移空出的位置用符号位填充，对于正数就是用0填充，对于负数则是用1填充

这种位移也被称为算数位移.
